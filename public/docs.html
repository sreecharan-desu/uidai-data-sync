Request Parameters

        
Universal parameters accepted in the request body.

        

            
                

                    Parameter
                    Type
                    Description
                

            
            
                
datasetStringRequired. One of enrolment, biometric, demographic.

                
limitNumberMax records to fetch (Max: 1000, Default: 100).

                
pageNumberPagination offset (Default: 1).

                
filtersObjectKey-value pairs for specific field matching (e.g., {"state": "Delhi"}).

Example Requests

        
Copy and paste these payloads into the Query Lab above.


        
1. Enrolment (All Parameters)

        
{
  "dataset": "enrolment",
  "limit": 10,
  "page": 1,
  "filters": {
    "state": "Maharashtra",
    "district": "Pune"
  }
}


        
2. Biometric Data (Simple)

        
{
  "dataset": "biometric",
  "limit": 5
}


        
3. Demographic Data by District

        
{
  "dataset": "demographic",
  "filters": { "district": "Bangalore" },
  "limit": 5
}
