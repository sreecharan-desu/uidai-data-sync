# CSV DOWNLOADS

Direct access to full and year-wise CSV datasets.

## 1. Full Dataset
Method: **GET**

```bash
https://uidai.sreecharandesu.in/api/datasets/biometric
https://uidai.sreecharandesu.in/api/datasets/enrolment
https://uidai.sreecharandesu.in/api/datasets/demographic
```

## 2. Split Dataset (Year-wise)
Method: **GET**

```bash
# 2025 Datasets
https://uidai.sreecharandesu.in/api/datasets/biometric?year=2025
https://uidai.sreecharandesu.in/api/datasets/enrolment?year=2025
https://uidai.sreecharandesu.in/api/datasets/demographic?year=2025

# 2026 Datasets
https://uidai.sreecharandesu.in/api/datasets/biometric?year=2026
https://uidai.sreecharandesu.in/api/datasets/enrolment?year=2026
https://uidai.sreecharandesu.in/api/datasets/demographic?year=2026
```

**Supported Datasets:** biometric, enrolment, demographic
